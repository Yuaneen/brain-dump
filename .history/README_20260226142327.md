# brain-dump

个人学习与成长记录库：技术、编程、架构、读书、英语与随想随笔。

## 目的

- **每日流水**：在 `daily-log/` 按日期记录当天所学与所想
- **专题沉淀**：在 `topics/` 对日常内容做提炼与深度笔记
- **代码复用**：在 `snippets/` 存放可复用的代码片段
- **资源归档**：在 `resources/` 存放链接、图片等外部资源
- **模板**：在 `templates/` 使用统一模板写日记与读书笔记

## 更新频率

尽量每日更新 `daily-log/YYYY-MM-DD.md`，专题与片段按需更新。

## 目录结构

```
daily-log/     # 每日流水账 (YYYY-MM-DD.md)
topics/        # 专题笔记：programming, architecture, english, books
snippets/      # 代码片段库
resources/     # 外部资源、图片等
templates/     # 笔记模板
```
